# 🏐 Volleyball Club Management System

A GUI-based desktop application to manage operations for a volleyball club, including handling locations, personnel, members, family relations, payments, team formations, email logs, and predefined reports. Built with **Python (Tkinter)** and **MySQL**.

---

## 📦 Features

- ✅ **Location Management** – Add club or event locations.
- 👤 **Personnel Module** – Register coaches, staff, etc.
- 🧑‍🤝‍🧑 **Family Member Management** – Add emergency contacts.
- 🎽 **Club Member Management** – Store and manage player info.
- 🤝 **Team Formation** – Assign members to teams.
- 💳 **Payments** – Record membership payments or donations.
- 📊 **Predefined Queries** – Run built-in reports (e.g., unpaid members).
- 📧 **Email Logs** – View communication logs (Subject, To, From, Body).
- 🗃️ **MySQL Backend** – Central database to persist data.

---

## 🛠 Technologies Used

- **Python 3**
- **Tkinter (GUI)**
- **MySQL (Backend)**
- **MySQL Connector/Python**
- **VS Code** (recommended)
